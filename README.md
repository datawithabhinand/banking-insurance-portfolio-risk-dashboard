# Banking & Insurance Portfolio Risk Dashboard

This project analyzes customer financial profiles to identify portfolio risk, churn probability, and customer segmentation patterns across banking and insurance products.

## Tools Used
- Power BI
- PostgreSQL
- SQL
- Data Analytics

## Key Insights
- Medium value customers show the highest churn percentage.
- High risk customers represent a significant portion of the portfolio.
- Insurance product distribution highlights strong Auto and Home policy concentration.
- Customer segmentation reveals mid-tier balance dominance.
